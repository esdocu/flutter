Flutter es un framework de código abierto de Google para crear hermosas aplicaciones multiplataforma compiladas de forma nativa a partir de una única base de código.

  • Rápido
  • Productivo
  • Flexible

Rápido

El código Flutter se compila en código de máquina ARM o Intel, así como en JavaScript, para un rendimiento rápido en cualquier dispositivo.

Pruébalo en DartPad

Productivo

Construye e itera rápidamente con Hot Reload. Actualiza el código y observa los cambios casi al instante, sin perder el estado.

Flexible
